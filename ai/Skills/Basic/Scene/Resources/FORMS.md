# 场景系统表单定义

## 场景类 (Scene) 表单

### 创建场景

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `name` | `std::string` | 场景名称 | 是 | - |
| `uuid` | `std::string` | 场景唯一标识符 | 否 | "" |

### 根节点获取

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| - | - | 根节点由引擎自动创建，调用 `getRoot2D()` / `getRoot3D()` 直接获取 | - | - |

## 节点基类 (Node) 表单

### 基本属性

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `name` | `std::string` | 节点名称 | 是 | - |
| `uuid` | `std::string` | 节点唯一标识符 | 否 | "" |
| `groupID` | `int` | 节点组 ID | 否 | 0 |

### 变换操作

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `position` | `Vec3` | 节点位置 | 否 | (0, 0, 0) |
| `worldPosition` | `Vec3` | 节点世界位置 | 否 | (0, 0, 0) |
| `rotation` | `Quat` | 节点旋转 | 否 | (0, 0, 0, 1) |
| `worldRotation` | `Quat` | 节点世界旋转 | 否 | (0, 0, 0, 1) |
| `scale` | `Vec3` | 节点缩放 | 否 | (1, 1, 1) |
| `worldScale` | `Vec3` | 节点世界缩放 | 否 | (1, 1, 1) |
| `eulerAngles` | `Vec3` | 节点欧拉角 | 否 | (0, 0, 0) |

### 节点层次结构

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `child` | `Node*` | 子节点 | 是 | - |
| `parent` | `Node*` | 父节点 | 是 | - |
| `childName` | `std::string` | 子节点名称 | 是 | - |

### 激活状态

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `active` | `bool` | 是否激活 | 是 | true |

### 组件管理

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `componentName` | `std::string` | 组件名称 | 是 | - |
| `componentUuid` | `std::string` | 组件唯一标识符 | 否 | "" |

## 2D 节点类 (Node2D) 表单

### 创建 2D 节点

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `name` | `std::string` | 节点名称 | 是 | - |
| `uuid` | `std::string` | 节点唯一标识符 | 否 | "" |

### 2D 特有属性

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `anchor` | `Vec2` | 节点锚点 | 否 | (0, 0) |
| `size` | `Size` | 节点大小 | 否 | (0, 0) |
| `angle` | `float` | 节点旋转角度 | 否 | 0.0f |

### 碰撞检测

| 字段名 | 类型 | 描述 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `x` | `float` | 碰撞点 X 坐标 | 是 | - |
| `y` | `float` | 碰撞点 Y 坐标 | 是 | - |
