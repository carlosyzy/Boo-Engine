# 事件系统表单定义

## 注册事件监听

| 参数 | 类型 | 描述 | 必填 |
|------|------|------|------|
| `eventName` | `std::string` | 事件名称 | 是 |
| `callback` | `std::function<void()>` | 回调函数（无参数） | 是 |

**返回值**：`uint64_t` 监听器 ID，用于后续取消监听

## 取消监听（按 ID）

| 参数 | 类型 | 描述 | 必填 |
|------|------|------|------|
| `id` | `uint64_t` | 监听器 ID | 是 |

## 取消监听（按事件名）

| 参数 | 类型 | 描述 | 必填 |
|------|------|------|------|
| `eventName` | `std::string` | 事件名称，取消该事件的所有监听 | 是 |

## 触发事件

| 参数 | 类型 | 描述 | 必填 |
|------|------|------|------|
| `eventName` | `std::string` | 事件名称 | 是 |
